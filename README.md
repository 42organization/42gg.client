<h2>프로젝트 소개</h2>

42 서울 내에서 탁구 경기 매칭, 전적 서비스를 제공하는 프로젝트 입니다.

🔸 Kakao 계정으로 로그인한 유저는 **매칭/전적 서비스**를 이용할 수 없습니다! 🔸

🔹 42 계정으로 로그인한 유저는 모든 기능 이용 가능! 🔹
